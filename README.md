# Dockerfile: Ubuntu Bionic, CMake, Qt 5.9

Dockerfile for creating images for building and automated testing of Qt 5 applications.

Image configuration:
- Ubuntu Bionic (18.04)
- Qt 5.9.5
- GCC/G++ 7.3.0
- CMake 3.10.2
- Doxygen 1.8.13
- gcovr 3.4.1
- Tags files for Qt 5.9.5 (to link in Doxygen documentation)
